В игре существует три типа юнитов - Конница, пехота и так называемые ниндзя-воины. У каждого юнита есть свое уникальное оружие - копье, меч и magic-Stick. Юниты созданы с помощью Абстрактной фабрики. Юниты состоят в отрядах, использован паттерн Компановщик. Отряды также могут состоять из других отрядов поменьше. Игра осуществляется с помощью объектов класса Player, у каждого игрока есть определнная сумма денег, на которые он может купить те или иные отряды воинов.
